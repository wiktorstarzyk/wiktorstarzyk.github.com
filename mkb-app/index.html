My Knowledge Base

My Knowledge Base is an app to store anything you want to remember, any piece of data that you may want to reference in the future. It is partly a note taking app and partly a personal database. It's great for those who have a hard time remembering things.

Data is organized into Entries that can store a variety of types of data including: text, integers, decimal numbers, boolean values, dates, photos, pencil drawings, web links, etc. Every piece of data supports an optional key(label), which makes it easier to find the exact piece of data that you are looking for.

My Knowledge Base also comes with a flexible query system that lets you quickly find exactly what you are looking for. Custom Entry Types/Templates allow you to input data as quickly and efficiently as possible and add more flexibility when querying data.
